# TextureEditor 贴图编辑器

## 🌟 功能介绍

TextureEditor 是一个贴图检查和编辑工具，用于快速预览和调整模型的贴图效果。

## 🛠️ 使用方法

1. 点击 **贴图编辑器** 按钮打开工具界面
2. 选择需要检查的模型
3. 工具会自动显示模型的贴图信息
4. 可以调整贴图的显示参数，如亮度、对比度等
5. 支持快速切换不同的贴图通道

## 🎯 特点

- **实时预览**：实时显示贴图效果，所见即所得
- **多通道支持**：支持漫反射、法线、高光等多种贴图通道
- **参数调整**：可以调整贴图的各种参数
- **快速切换**：在不同贴图之间快速切换
- **贴图检查**：检查贴图是否正确应用

## 💡 提示

- 使用 TextureEditor 可以快速发现贴图问题
- 调整贴图参数时，建议在不同角度观察效果
- 对于复杂模型，可以分区域检查贴图
- 结合模型的 UV 布局使用，效果更佳
