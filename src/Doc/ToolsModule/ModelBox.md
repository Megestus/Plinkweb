# ModelBox 模型库工具

## 🌟 功能介绍

ModelBox 是一个模型库管理工具，用于快速访问和管理常用的模型资源。

## 🛠️ 使用方法

1. 点击 **模型库** 按钮打开工具界面
2. 在模型库中浏览和搜索可用的模型
3. 选择需要的模型并点击导入按钮
4. 模型会自动添加到当前场景中

## ⚙️ 配置

模型库的存储页面中设置模型库的存储路径（建议共享盘）

<br />

## 储存

也在存储页面中&#x20;

先选择存储文件夹后

选中模型-点击刷新选择-截图-导出模型-保存 即可

## 🎯 特点

- **快速访问**：一键打开模型库，快速找到所需模型
- **分类管理**：按类别组织模型，方便查找
- **自定义配置**：根据项目需求调整模型库设置
- **批量导入**：支持同时导入多个模型

## 💡 提示

- 建议将常用模型放在模型库中，提高工作效率
- 定期更新模型库，保持资源的新鲜度
- 使用有意义的命名方式组织模型，便于搜索
